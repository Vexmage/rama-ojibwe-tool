# Rama Ojibwe Tool

A small study-oriented Python project for exploring Rama-centered Ojibwe vocabulary, phrase patterns, and limited conjugation logic.

## Phase 1 Goals

- Create a clean local CLI project
- Store a very small set of manually entered lexical items
- Load and display entries from JSON
- Prepare for later rule-based conjugation work
- Keep the project easy to upload and maintain on GitHub

## Initial Scope

This is a personal learning tool, not an authoritative language resource.
It will begin with a very small curated dataset drawn from Rama reference materials.

## Planned Structure

- `data/` lexical entries
- `notes/` grammar and source notes
- `scripts/` helper utilities
- `app.py` main CLI entry point
